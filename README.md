# EduGate – Online Courses Management Platform

## Project Description
EduGate is an online learning management platform built using Django. The system supports multiple user roles including Admins, Teachers, Students, and Parents. The platform enables teachers to create courses and lessons, students to enroll and learn, parents to monitor student progress, and admins to manage the entire platform. EduGate is designed with clean architecture, responsive UI, and modern best practices using Django templates, static files, media handling, and cloud storage for large assets.

## Features
- User Authentication (register, login, logout)
- User Authorization with multiple roles (Admin, Teacher, Student, Parent)
- Course creation, editing, publishing, and management
- Lesson and module management inside each course
- Student enrollment system
- Assignments creation, submission, and grading
- Parent dashboard to monitor child progress and grades
- Admin dashboard for platform-wide management
- Responsive UI using Bootstrap
- Template inheritance for clean and reusable layouts
- Dynamic URLs, static files, and media handling
- Cloud storage support for files (Firebase)
- Environment variables using python-dotenv
- Pagination, feedback messages, and user-friendly navigation
- Git & GitHub workflow with daily commits, forks, and pull requests

## User Stories for the EduGate Platform:
- As a Student, I can register an account so that I can enroll in courses.  
- As a Student, I can log in to view my enrolled courses and lessons.  
- As a Student, I can enroll in a course so that I can access its modules and lessons.  
- As a Student, I can submit assignments so that teachers can grade my work.  
- As a Student, I can view my grades and progress so I can track my performance.  
- As a Teacher, I can create courses so that students can enroll in them.  
- As a Teacher, I can add modules and lessons to my courses so I can structure my learning content.  
- As a Teacher, I can create assignments so that students can complete their tasks.  
- As a Teacher, I can grade student submissions so I can evaluate their performance.  
- As a Parent, I can view the courses my child is enrolled in so I can monitor their learning.  
- As a Parent, I can view my child’s progress and grades so I can support their education.  
- As an Admin, I can manage all users to ensure the platform runs smoothly.  
- As an Admin, I can approve or reject newly created courses to control published content.  
- As an Admin, I can access a dashboard showing platform statistics for oversight.
